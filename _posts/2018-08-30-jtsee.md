### jtsee.com 建立
-----------------    

* aliyun ECS云服务
* git
* nodejs

### 步骤
-----------------

1. 连接 aliyun    
	* ssh   
2. git 同步到aliyun
	* 本地git
	```
	   git add
	   git commit -m
	   git push
	```
	* 服务器配置
	```
	git init --bare jtsee.git   //--bare 是代表创建裸仓库
	//创建web站点目录www
	//自动同步功能用到的是 git 的钩子功能
	cd git/jtsee.git
	cd hooks
	//这里我们创建post-receive文件
	vim post-receive
	//在该文件里输入以下内容
	#!/bin/bash
	git --work-tree=/root/www checkout -f
	//保存退出后，将该文件用户及用户组都设置成git
	//由于该文件其实就是一个shell文件，我们还应该为其设置可执行权限
	chmod +x post-receive
	```
