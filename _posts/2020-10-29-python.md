### python 包管理

#### pip
pip Python 包安装和管理工具

pip3 --version    # Python3.x 版本命令

##### 升级 pip：
```
pip install -U pip
```

##### 升级包

pip install --upgrade SomePackage

##### 卸载包

pip uninstall SomePackage

##### 本地项目打包安装

pip install -e 项目根目录
